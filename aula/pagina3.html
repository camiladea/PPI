O CSS

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar elementos em uma página da web. Com isso, é possível separar o conteúdo (HTML) da sua representação visual (CSS).

Surgimento: O CSS foi proposto por Håkon Wium Lie em 1994 para resolver a complexidade da formatação HTML da época.
Desenvolvimento: Trabalhando com o W3C, a linguagem foi desenvolvida e padronizada, com o CSS1 sendo lançado em 1996.
Evolução: Posteriormente, foram lançados o CSS2 (1998) e o CSS3 (2012), com novas funcionalidades e recursos para o desenvolvimento web.
Objetivo: O CSS visa separar a estrutura do conteúdo (HTML) de sua apresentação visual (CSS), facilitando a manutenção e o desenvolvimento de sites.
